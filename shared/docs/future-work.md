# Future Work

## OpenAI Integration

Replace rule-based resume interpretation, matching explanations, interview question generation, and answer scoring with configurable OpenAI-backed services behind the existing AI interfaces.

## Embeddings and pgvector

Add semantic matching using embeddings stored in PostgreSQL with `pgvector`, or use ChromaDB for local vector search.

## Real-Time Notifications

Introduce WebSocket or server-sent event notifications for application updates, interview completion, and recruiter review actions.

## Email Invitations

Add email-based candidate invitations, interview reminders, and application status notifications.

## Interview Audio and Video

Extend the interview simulator with recorded audio/video responses, transcript generation, and multimodal evaluation.

## Recruiter Teams

Support organizations, recruiter teams, shared jobs, reviewer assignments, and team-level permissions.

## Bias and Fairness Checks

Add explainability controls, fairness audits, scoring transparency, and bias monitoring for AI-assisted recommendations.

## Deployment

Prepare optional production deployment using managed PostgreSQL, HTTPS, environment-specific settings, CI checks, and container registry workflows.
